## The scientific bottom line

Two independent gold-free instruments — one static, one executing — both reach **`0/10`** on the
plausible-but-wrong-value class, with **no missing outcomes** and a Wilson upper bound of `0.278`. That is the
mitigation ceiling, measured rather than argued.

Once the trivially-separable patches are excluded, the two instruments have **indistinguishable recall and
indistinguishable false-alarm rates**. Neither approach is better; both are blind to the same class.

## Four things a successor must not misread

1. **The benchmark has a severe gold-lookup shortcut.** `29` of `54` negatives are byte-identical to public
   gold; `0` of `13` positives are. A detector consulting gold scores `13/13` recall understanding nothing.
   Compare detectors on the `25`-row hard subset (`score_iter233_release.py --stratify`), never on the full
   negative set. This is the mission's own thesis turned on its own artifact.
2. **The stage B import probe is withdrawn.** Running imports in isolation false-alarms on
   configuration-dependent frameworks — it rejected 6 rows and 5 rejections were wrong. Replaced by: a row is
   an instrument failure iff it exited non-zero having printed no `RESULT=`. Do not resurrect the probe.
3. **`divergence_type` is witness-specific, not patch-intrinsic.** A patch labelled `crash_or_type` returned a
   clean value under an independently generated exercise. Even the crash class is only partially reachable
   without a reference.
4. **`65/67` coverage is deliberate.** The two `psf/requests` rows need an HTTP client on the forbidden list.
   Holding that line is the point; they stay uncovered.

  value class is `0/10` across two instruments with complete data, and the crash class is only partially

- **Mechanism confirmation** — the `p=0.008` fix-complexity lead, on a held-out cohort.
- **Benchmark hardening** — if the benchmark is to be used seriously, the gold-identical negatives are its
  biggest weakness. Building a negative set that is *not* trivially separable would require new
  certified-correct patches that differ from gold, which means new solver runs and provider spend.

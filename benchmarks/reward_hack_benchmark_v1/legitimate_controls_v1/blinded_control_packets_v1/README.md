# Blinded Legitimate-Control Packets v1

This directory contains prompt-ready code-review packets for the paired legitimate controls of the
historically misnamed v1 artifact. Iter192 established that the paired candidate rows are suite-failing patches,
not reward hacks. This packet artifact itself includes no model outputs, scores, rankings,
provider calls, source labels, gold/control labels, official reports, or prior verdicts.

Each packet exposes only:

- repository and instance id,
- public SWE-bench task text,
- public target-test identifiers,
- candidate diff,
- neutral review instruction and required JSON output shape.

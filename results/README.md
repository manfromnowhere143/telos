# Results

Canonical results are published in each experiment's version-controlled, correction-preserving `RESULT.md`
and proof directory, rather than duplicated here. The current standing result and correction ledger are
summarized in the root `README.md`; the manuscript source and status are in `paper/`.

The active gate is iter202, which remains a pre-result amended protocol with no retained or inspected solver output;
interrupted provider contact preceded its first Git freeze. Do not publish or
summarize an iter202 result claim until its `RESULT.md`, proof artifacts, all eight execution-shard
receipts, the single-run-attempt aggregate receipt, and validation guards have been created and validated
as one evidence unit. The receipt chain must bind the repository, workflow, run, attempt, commit, runtime
manifest, spec index, and every retained execution log; partial or mixed-attempt evidence is invalid.

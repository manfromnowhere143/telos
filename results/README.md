# Results index

This directory is not a results authority and does not duplicate empirical
findings. Canonical experiment outcomes live beside their preregistrations in
each `experiments/iter*_*/RESULT.md` and proof directory.

Use these current entrypoints:

- [`mission/current.json`](../mission/current.json) for the mutable operational
  pointer;
- [`README.md`](../README.md) for the bounded current scientific summary;
- [`docs/EXPERIMENT_INDEX.md`](../docs/EXPERIMENT_INDEX.md) for the complete
  correction-preserving experiment ledger;
- [`paper/telos.tex`](../paper/telos.tex) for the findings manuscript.

The root `HANDOFF.md`, `CONTINUITY.md`, and `mission/loop.json` are protected
historical artifacts. Their older “current” wording is chronology, not present
authority.

Do not add a new finding, rate, score, cost, run status, active gate, or next
action here. Add it to a separately authorized experiment record, propagate it
through the canonical current claim surfaces, and let the claim registry prove
that propagation.
